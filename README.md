# Scientific Report Assistant

An AI-powered web application designed to assist students in drafting scientific reports, featuring a robust gap-analysis review system.

## Features

-   **AI Draft Review**: Upload .docx, .pdf, or .txt files to receive a detailed criterion-by-criterion gap analysis.
-   **Structured Editor**: Guided writing sections based on standard scientific report structure (Introduction, Methods, Results, Discussion).
-   **Marking Criteria Integration**: Built-in rubric derived from "NAT11225026 Assessment 1 Part 2" to ensure all requirements are met.
-   **Nova Chat**: Context-aware AI chat assistant.

## Tech Stack

-   **Frontend**: HTML5, TailwindCSS (via CDN).
-   **Libraries**:
    -   `mammoth.js` (Docx processing)
    -   `pdf.js` (PDF processing)
    -   `font-awesome` (Icons)

## Usage

1. Open `index.html` in a browser.
2. Enter your name.
3. Upload a draft report to test the AI Review feature.
   *(Note: The AI features require a specific environment loop to function fully. In standard browsers, a mock mode is active for demonstration.)*
